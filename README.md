# Requisitos previos
Para poder utilizar la app es necesario:\\
-Tener instalada la app **RStudio**\\
-Tener instalados los paquetes **shiny** y **bslib**. (install.packages("shiny");install.packages("bslib"))

# Intsrucciones de ejecución
1. Descargar el archivo **App.rRmd**
2. Abre el archivo utilizando RStudio
3. Asegúrate de tener instalados los paquetes anteriores.
4. Haz click en la pestaña **Run->Run All** (Ctrl+Alt+R)
